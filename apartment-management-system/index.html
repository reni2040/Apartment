Apartment Management System

A comprehensive solution for society management

For Owners

  • View & pay maintenance fees
  • Download payment reports
  • Raise complaints
  • View society notices

For Management

  • Set & update maintenance fees
  • Manage users & permissions
  • Track payments & generate reports
  • Handle complaints & notices

User Roles

Admin

Full control

Owner

Pay fees, view reports

President

View reports & notices

Secretary

Manage notices & complaints

Treasurer

Manage payments & reports

Executive

Limited access

To use this system, you need to run it on a PHP server with MySQL.

Installation Required: This is a PHP application that requires a MySQL database. Run php -S localhost:8000 in the project directory, then visit http://localhost:8000/install to set up the system.
